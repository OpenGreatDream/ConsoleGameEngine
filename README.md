### What is ConsoleGameEngine(CGE)?

ConsoleGameEngine is a game engine for making text-based/pixel-based games in windows console, to a certain extent it was inspired by [olcConsoleGameEngine](https://github.com/OneLoneCoder/videos/blob/master/olcConsoleGameEngine.h).
